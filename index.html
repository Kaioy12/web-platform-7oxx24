Resumo da aula do dia 17/05

Primeiro aprendemos os conceitos básicos da programação, como:

  • Linguagem compilada: Em uma linguagem compilada, a máquina de destino traduz o programa diretamente
  • Linguagem interpretada: Em uma linguagem interpretada,o código fonte não é traduzido diretamente pela máquina de destino. Em vez disso, um programa diferente, o interpretador, lê e executa o código.
  • Cliente:Um servidor é um host que está executando um ou mais serviços ou programas que compartilham recursos com os clientes.
  • Servidor:Um cliente não compartilha qualquer de seus recursos, mas solicita um conteúdo ou função do servidor. Os clientes iniciam sessões de comunicação com os servidores que aguardam requisições de entrada. Além disso, também aprendemos algumas tags de html e a como organiza-las para formar uma imagem, texto, parágrafo,etc.
